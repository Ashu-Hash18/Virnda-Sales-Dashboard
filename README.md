Vrinda Store Sales Dashboard (Excel Project)
📊 Overview

Developed an interactive Excel dashboard to analyze sales data and generate actionable business insights for Vrinda Store.

🎯 Objectives
Analyze customer demographics and purchasing behavior
Identify top-performing regions and sales channels
Track overall sales performance
🔍 Key Insights
Female customers drive ~65% of total sales
Maharashtra, Karnataka, and Uttar Pradesh contribute ~35%
Age group 30–49 accounts for ~50% of sales
E-commerce platforms (Amazon, Flipkart, Myntra) contribute ~80%
🛠 Tools & Techniques
Microsoft Excel
Pivot Tables & Pivot Charts
VLOOKUP / XLOOKUP
Data Cleaning & Transformation
Interactive Dashboard (Slicers)
📈 Business Impact

This dashboard helps in identifying target customers, optimizing regional strategies, and improving channel performance for better decision-making.
